# FinAlly — Bug Tracker

Log of bugs encountered during development, their root cause, and resolution.

Format per entry:
- **ID**: BUG-NNN
- **Date**: YYYY-MM-DD
- **Section**: Plan section where the bug appeared (e.g., §8 API, §10 Frontend)
- **Symptom**: What was observed
- **Root cause**: Why it happened
- **Resolution**: How it was fixed
- **Status**: Open / Resolved
